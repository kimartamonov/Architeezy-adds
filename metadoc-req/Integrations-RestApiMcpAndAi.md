# Требования к странице: Как использовать REST API, MCP и AI поверх кастомной метамодели

## Раздел
Интеграция и автоматизация

## Страница
Как использовать REST API, MCP и AI поверх кастомной метамодели

## Назначение страницы
Показать, как пользовательские метамодели могут использоваться в автоматизации, AI-сценариях и интеграциях через API и MCP.

## Для кого эта страница
Архитекторы, разработчики автоматизации, platform teams и authors of AI workflows.

## Что должно быть написано на странице
- типовые сценарии чтения и обновления моделей через API
- как проектировать язык с учётом будущей автоматизации
- какие задачи AI хорошо решает поверх метамодели
- примеры use cases: каталог контрактов, сервисный реестр, data product каталог, AI-навигация по модели

## На какие вопросы пользователя страница должна отвечать
- Как использовать мою метамодель через API?
